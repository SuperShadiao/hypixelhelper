# 小沙雕的Mod帮助

## Hypixel Helper Mod安装

1. 安装forge
    - 如果你使用官启：
       - 首先启动一次1.8.9原版，然后下载群文件里的forge安装器运行，选择对应游戏目录进行安装（一般是默认的），然后重启启动器后启动一遍游戏，此时会在游戏目录生成mods文件夹
    - 如果你使用pcl，hmcl等第三方启动器：
       - 这些启动器一般在下载时会自动帮你装forge。（记得forge版本一定要最新版的2318，并且不要试图在forge里内置optifine，会出很多bug，optifine使用群内mod pack的optifine，直接拖入mods文件夹即可）
2. 安装mod并进入游戏
   1. 在群内下载Hypixel Helper Mod jar文件 (或者[点击这里](https://xiaoshadiao.club/hh))
   2. 扔入mods文件夹（pcl等第三方启动器可以直接拖到启动器对应的mod列表窗口）
   3. 直接启动游戏即可
   4. 可选择性的添加群内mod pack里的mod，小沙雕上传的mod没有冲突，当然如果你有自己的mod搭配的话也没事。
3. 本Mod注意事项
    - Hypixel Helper只能在forge上使用, 无法在blc, lunar和非windows系统 (手机, linux mac等) 上使用
    - 实际上有些人喜欢叫这个mod为沙雕端 (被看做是一个端), 但其实质是一个模组, 底层是forge加载, 所以我更愿意称其为forge端
    - 而且, 就算能装, "端中端"一定会崩溃, 比如feather端, 之前一起装会导致本Mod加载界面错误发生崩溃, 经过我修复之后又因为我Mod自定义背景问题导致feather替换材质包时崩溃, 最后还出现了渲染错误问题而放弃兼容

## SkyDiao Mod安装

1. 安装Fabric
    - 如果你使用官启：
       - 首先启动一次1.21.11原版，然后下载群文件里的Fabric安装器运行，选择游戏版本1.21.11直接安装，然后重启启动器后就可以看到Fabric启动配置了，此时启动游戏后会在游戏目录生成mods文件夹
    - 如果你使用pcl，hmcl等第三方启动器：
       - 这些启动器一般在下载时会自动帮你装Fabric。选中1.21.11安装Fabric，然后将mod安装即可
2. 安装mod并进入游戏
   1. 在群内下载SkyDiao Mod jar文件 (或者[点击这里](https://5ixsd.top/skydiao))
   2. 扔入mods文件夹（pcl等第三方启动器可以直接拖到启动器对应的mod列表窗口）
   3. 直接启动游戏即可
   4. 可选择性的添加群内mod pack里的mod，小沙雕上传的mod没有冲突，当然如果你有自己的mod搭配的话也没事。
3. 本Mod注意事项
    - 本Mod比较灵活, 可以在多种设备/系统上 (包括手机) 使用.
